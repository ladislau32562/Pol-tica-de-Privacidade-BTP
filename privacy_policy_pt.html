Política de Privacidade — Comptage Travail

Data de vigência: 15/09/2025

Esta Política descreve como a aplicação Comptage Travail trata os dados do utilizador.

1. Dados que processamos

  • Dados de registos: data, estado (pendente/aprovado), chefe, local, observação, horas suplementares, turno (dia/noite), uso de marteau piqueur e respetiva “prime”.
  • Dados técnicos: versão da aplicação e informações básicas de diagnóstico (apenas locais, sem envio para servidores).

2. Origem e finalidade

Os dados são introduzidos manualmente pelo utilizador e usados exclusivamente para gestão de ponto e cálculo de horas.

3. Armazenamento e retenção

  • Todos os dados são guardados localmente no dispositivo (SQLite).
  • Não transferimos dados para servidores de terceiros.
  • O utilizador pode eliminar ou editar registos a qualquer momento.

4. Partilha

Não partilhamos dados com terceiros. Não exibimos anúncios.

5. Permissões

Atualmente a app não solicita permissões sensíveis. Se, no futuro, forem necessárias permissões (ex.: armazenamento para exportar CSV), serão pedidas com explicação clara.

6. Direitos do utilizador

  • Aceder, corrigir, eliminar dados locais.
  • Exportar registos (quando disponível) para formatos de texto/CSV.

7. Segurança

Os dados permanecem no dispositivo do utilizador. Recomendamos proteger o dispositivo com PIN/biometria.
